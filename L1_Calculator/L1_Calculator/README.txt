CALCFLOW - PROFESSIONAL CALCULATOR

A responsive calculator project built with HTML, CSS Grid, and JavaScript.

FILES:
- index.html
- style.css
- script.js

FEATURES:
- Addition, subtraction, multiplication and division
- Percentage calculations
- Decimal numbers
- AC and DEL controls
- Calculation history
- Keyboard support
- Responsive professional UI
- CSS Grid button alignment

KEYBOARD:
Numbers / Operators = Input
Enter or = = Calculate
Backspace = Delete
Escape = Clear

Open index.html in your browser to run the project.
